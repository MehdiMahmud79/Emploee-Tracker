  To start you may need to run the following command and answer the prompt question with an approperiate answer.

## questions
You will be asked to do:
```
  * view all departments
  * view all roles
  * view all employees 
  * add a department
  * add a role
  * add an employee
  * update an employee role
  * exit

```
